# 김치옥 MOT 점검 웹앱

매장에서 스마트폰으로 **MOT(Moment of Truth) 서비스 체크리스트**를 점검하고, 결과를 **구글 스프레드시트로 자동 전송**하는 모바일 웹앱입니다. 별도 설치 없이 URL 접속만으로 동작합니다.

🥬 **김치옥 전용** (체크리스트 v3 · MOT 7단계 35문항) · 고반식당은 별도 앱으로 운영합니다.

## ✨ 주요 기능
- 주문방식(직접주문/테이블오더)에 따른 점검 항목 자동 분기
- ○ 이행 / △ 미흡 / × 불이행 3단 평가 + **문항별 가중 배점**(미흡=배점 절반, 불이행=배점) → **100점 만점 환산**
- 총점·등급(S 90%↑ / A 80~89% / B 70~79% / C 70% 미만) 자동 계산
- **점검율(점검 커버리지)** 표시 — 해당없음·미응답을 뺀 실제 점검 문항 비율(예: 26 / 40 = 65%, 분모는 선택한 주문방식 기준)
- **점검율 등급 상한** — 80% 이상 제한 없음 / 60% 이상 A / 50% 이상 B / 50% 미만 C(재점검 계획 대상). 최종 등급은 점수 등급과 상한 중 낮은 쪽
- 🚫 절대 금지 10계명(위반 1건당 −5점)
- 결과를 구글 스프레드시트(`점검요약`·`항목상세`)에 누적 저장
- 오프라인 점검 → 온라인 복귀 시 재전송, CSV 내보내기, 점검 이력
- 진행바·단계 네비게이션 상단 고정, 다크모드, 버전·변경이력 표시

## 📁 구성
| 파일 | 설명 |
|------|------|
| `index.html` | 모바일 웹앱 본체 (단일 파일, 빌드 불필요) |
| `google-apps-script.gs` | 구글 시트 수신 스크립트 (Apps Script) |
| `사용설명서.md` | 설치·연동·사용 상세 가이드 |
| `김치옥_MOT_체크리스트_v3.xlsx` | 원본 체크리스트 (v3) |

## 🚀 사용 방법
1. `index.html`을 웹 호스팅(GitHub Pages/Netlify 등)에 올리거나 브라우저로 열기
2. 기본 구글 시트 연동 URL이 앱에 포함되어 있어 별도 설정 없이 결과가 전송됩니다. 다른 시트로 보낼 때만 앱 **설정 → Apps Script 배포 URL**을 변경하세요.

### GitHub Pages로 배포하기
저장소 **Settings → Pages → Branch: `main` / root** 선택 후 저장하면
`https://everside76.github.io/kim_check/` 주소로 바로 접속할 수 있습니다.
