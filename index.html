    직원 휴대폰 홈 화면에 앱 아이콘을 추가하면 일반 앱처럼 바로 실행할 수 있습니다.

    ⚠️ 카카오톡·메신저로 받은 링크는 먼저 Chrome / Safari로 열기 후 진행하세요. (메신저 인앱 브라우저는 홈 화면 추가가 제한됩니다)
    iPhone Safari: 공유 버튼 → 홈 화면에 추가
    Android Chrome: 오른쪽 위 메뉴 → 앱 설치 또는 홈 화면에 추가
    브라우저가 자동 설치를 지원하면 위의 바로가기 만들기 버튼으로 바로 추가할 수 있습니다.
